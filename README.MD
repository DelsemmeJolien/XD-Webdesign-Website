
# ✅ Checklist Eindopdracht Website (HTML & CSS)

## 1. Bestanden & structuur
- [X] Heb ik een **index.html** en minimaal 2 extra pagina’s?
- [X] Staat mijn CSS in een **aparte map** (`/css/style.css`)?
- [X] Zijn alle afbeeldingen in een **/images/** map geplaatst?
- [X] Is mijn mappenstructuur overzichtelijk en correct?

## 2. HTML
- [X] Gebruik ik **semantische tags** (header, nav, main, section, footer)?

- [X] Zijn alle links in de **navigatie** correct en werken ze?
- [X] Heb ik **alt-teksten** toegevoegd aan alle afbeeldingen?
- [X] Is mijn HTML **gevalideerd via [W3C Validator](https://validator.w3.org/aat alle styling in **style.css** (geen inline CSS)?

- [X] Heb ik **hover-effecten** op knoppen of menu-items?

---

## 4. Responsiviteit
- [X] Gebruik ik **flexbox** of **CSS grid** voor layout?
- [X] Heb ik minimaal **één media query** voor mobiele weergave?
- [X] Werkt mijn site goed op **desktop én mobiel**?

---

## 5. Afbeeldingen
- [X] Zijn afbeeldingen **geoptimaliseerd** (niet te groot)?
